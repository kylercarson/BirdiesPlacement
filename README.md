# BirdiesPlacement

This repository contains the website for Birdies Placement, a senior living and hospice placement service.

## Hosting on GitHub Pages

This site is designed to be hosted on GitHub Pages. To enable:

1. Go to your repository settings
2. Scroll down to "Pages" in the left sidebar
3. Under "Source", select "Deploy from a branch"
4. Choose the "main" branch and "/ (root)" folder
5. Click "Save"

Your site will be available at `https://kylercarson.github.io/BirdiesPlacement/`

## Local Development

To view the site locally, open `index.html` in your web browser.
